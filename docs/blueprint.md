# **App Name**: PlushyeSwL AI Enhanced

## Core Features:

- Message Sending: Allows users to send text messages to the PlushyeSwL AI.
- AI Response Generation: Generates a response to the user's message using the PlushyeSwL AI model.
- Clear Memory: Clears the PlushyeSwL AI's memory of previous interactions using a button, allowing the conversation to restart from scratch.
- Message History: Displays a history of messages between the user and the AI.
- Thinking Indicator: Provides a status bar and progress bar to show when the AI is processing a message and generating a response.
- Error Handling: Displays an error message if the AI is unable to connect to the backend.
- Response Time Display: Displays the time it took for the AI to generate a response.

## Style Guidelines:

- Primary color: Dark background, similar to the provided image, to create a modern and focused interface.
- Accent color: Subtle electric purple or similar vibrant color for interactive elements like the message input border and send button.
- Body font: A clean, sans-serif font like 'Inter' or 'Roboto' for readability on both desktop and mobile devices.
- Single-column layout optimized for mobile and desktop. Input area fixed at the bottom with a clearly defined text input field and send button.  Minimize clutter to maintain focus on the conversation.
- Use simple, outlined icons for attachment, microphone, and send functionalities in the input area. Ensure icons are easily recognizable.
- Subtle animation for the progress indicator when the AI is processing. A simple pulsing effect or expanding line can be used.
